Subscribe to alerts

When the earliest slot changes, an alert is posted to these public channels. There is no signup. Pick one or both.

  • Telegram channel Open in the Telegram app.
  • ntfy topic Install the ntfy app and subscribe to this topic for push notifications.

What this site does and does not do

  • This is an unofficial alert tool.
  • It does not book, change, cancel, or hold appointments.
  • Users must complete all changes on the official Service NSW website.
  • No booking number or family name is collected on this site.
  • No email, phone number, or personal identifier is collected.
  • Slot data is provided as-is, may be stale, and may be wrong. Always verify on Service NSW before relying on it.

Slot data is refreshed by a scheduled job. GitHub Actions schedules can run late or be paused, so freshness is best-effort.
